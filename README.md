# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using python, sql and power Bi.

# Overview
This project analyzes retail customer shopping behavior to help a business improve sales performance, customer engagement, and long term loyalty.
The objective was to understand purchasing patterns across demographics, product categories, discounts, reviews, shipping preferences, and subscription status. The analysis focuses on identifying key factors that influence customer decisions and repeat purchases.
The project combines Python for data analysis, PostgreSQL for structured querying, and Power BI for interactive visualization.

# Dataset
The dataset contains customer transaction level information, including:
  * Customer demographics (age group, gender)
  * Product details (category, item purchased)
  * Purchase amount
  * Review ratings
  * Discount applied (Yes/No)
  * Shipping type (Standard / Express)
  * Subscription status
  * Previous purchases
